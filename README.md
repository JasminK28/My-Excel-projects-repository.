# Excel Data Analysis Portfolio Project

## Overview

This project demonstrates my ability to use Microsoft Excel to analyse, organise, summarise, and visualise data. Throughout a series of practical exercises, I worked with retail sales and bike sales datasets to develop key data analysis skills including filtering, sorting, formulas, PivotTables, and data visualisation.

## Skills Demonstrated

* Data filtering and sorting
* Using Excel formulas including SUM, AVERAGE, and SWITCH
* Data exploration and analysis
* Creating and interpreting PivotTables
* Data visualisation using charts and graphs
* Identifying trends and patterns within datasets
* Presenting findings and insights

## Tasks Completed

### Retail Sales Data Analysis

Using a retail sales dataset, I:

* Filtered and sorted customer data.
* Used the SUM function to calculate total commission values.
* Used the AVERAGE function to calculate average commission.
* Explored the dataset to identify trends and gain familiarity with Excel's data analysis tools.

### Bike Sales Pivot Table Analysis

Using a bike sales dataset, I created PivotTables to analyse customer demographics and sales performance across different countries.

Key findings included:

* Germany's customers were primarily in the Adult (35–64) age group.
* Australia had sales across all customer segments and markets.
* Adult female customers generated the highest total sales.
* The United States showed the highest overall profitability.
* Adults generated the highest sales compared to other age groups.
* Canada showed stronger sales among young adult customers.

### Sales Performance Analysis by County

I created a PivotTable to summarise sales volumes by county and product type. I also used the SWITCH function to categorise products based on sales performance:

* High Sales: Greater than 600 units
* Medium Sales: Between 300 and 600 units
* Low Sales: Less than 300 units

This exercise improved my understanding of logical functions and data categorisation techniques.

### Data Visualisation

Using the bike sales dataset, I created charts and visualisations to present sales trends and customer insights in a clear and meaningful way. This helped develop my ability to communicate data findings visually.

## Conclusion

This project strengthened my Excel data analysis skills and provided practical experience working with real-world datasets. It demonstrates my ability to clean, analyse, summarise, and visualise data while drawing meaningful insights from business information.
